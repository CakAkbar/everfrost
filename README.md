### Run Locally

To run **Adex** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/naincode/adex-landing-page.git
```

Windows:

```bash
git clone https://github.com/naincode/adex-landing-page.git
```
